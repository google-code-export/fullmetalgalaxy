Les tirs se déroulent de la même façon que les contrôles, mais peuvent être effectués à distance :
les Chars et Vedettes ont une portée de 2 cases (3 sur une montagne)
le Gros tas a une portée de 3 cases.

Attention cependant, comme vous avez lu les règles, vous savez que :
- Il n'est pas possible de pénétrer dans une zone de feu adverse (case à portée de tir de deux destructeurs). Pour vous aider, vous pouvez afficher les zones de feux grâce au bouton
- Un destructeur ne peut tirer que deux fois dans un même tour (des marqueurs indiquent le nombre de munitions restantes).

A vous de :
  1. Détruire le premier char.
  2. Neutraliser le second char.
  3. Détruire la première tourelle de l'astronef rouge (avec votre Vedette et le gros tas).
  4. Controler le second char.
  5. Détruire la dernière tourelle.
